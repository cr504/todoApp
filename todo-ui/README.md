# todo-ui

## Project setup
Open a terminal in the todo-ui directory then run the following 
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

Go to *http://localhost:8080*

### Compiles and minifies for production
```
npm run build
```
